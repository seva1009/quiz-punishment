> "Aprender nunca fue tan arriesgado" ⏳

Una aplicación innovadora que combina aprendizaje gamificado con consecuencias reales. Responde preguntas correctamente o sufrirás bloqueos temporales en la app.

##  Características Destacadas

| Característica | Descripción |
|---------------|-------------|
| ** 50+ Preguntas** | Matemáticas, cultura, ciencia, historia, geografía |
| ** Castigos Progresivos** | Desde 1 minuto, aumenta con errores consecutivos |
| ** Sistema de Puntos** | +10 puntos por acierto, ranking por porcentaje |
| ** Historial Detallado** | Registro de todas tus respuestas y errores, en proximas versiones abra una seccion de "tiempo offline" |
| ** Interfaz Optimizada** | Colores, barras de progreso, diseño para móvil |
| ** Preguntas Aleatorias** | Mezcla diferente cada partida |
| ** juego ideal si quieres dejar la procrastinación y aprender al mismo tiempo** | 
| ** En futuras versiones, el teléfono quedará completamente bloqueado.** |

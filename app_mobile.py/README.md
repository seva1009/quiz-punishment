An innovative application that combines gamified learning with real consequences. Answer questions correctly or suffer temporary app blocks.

##  Key Features

| Feature | Description |
|---------|-------------|
| ** 50+ Questions** | Mathematics, culture, science, history, geography |
| ** Progressive Punishments** | From 1 minute, increases with consecutive errors |
| ** Points System** | +10 points per correct answer, ranking by percentage |
| ** Detailed History** | Record of all your answers and errors, In future versions, there will be an "offline time" section | 
| ** Optimized Interface** | Colors, progress bars, mobile design |
| ** Random Questions** | Different mix each game |
| ** Ideal game if you want to stop procrastinating and learn at the same time** |
| ** In future versions, the phone will be completely locked.** |
